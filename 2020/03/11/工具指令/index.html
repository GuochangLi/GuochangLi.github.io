Hexo

插入图片

  • 在站点配置文件中找到post_asset_folder这个属性，设置为true，这样在每次hexo new "article"的时候，会同时产生一个article文件夹。

  • 将图像放在这个article文件夹中，然后在文章中使用相对路径引用。

  • npm install https://github.com/CodeFalling/hexo-asset-image --save

  • 安装完成后再使用hexo g的时候，就可以看到图片已经自行加载上去了

PS：如果直接使用npm install hexo-asset-image --save，这样安装的版本似乎是与hexo-math有冲突，这样会导致保存(hexo插入图片是真的坑
