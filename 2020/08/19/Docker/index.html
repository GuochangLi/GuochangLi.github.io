Docker使用

镜像

获取镜像

  • 获取镜像使用docker pull [NAME]

启动镜像

  • docker run -it [NAME] [CMD]

-i表示交互式操作

-t表示终端

-d表示后台执行

-P表示将容器内部使用的网络端口随机映射到我们使用的主机上

CMD表示命令

  • 如果要退出终端可以直接输入exit

删除镜像

  • docker rmi [NAME]

查看镜像

  • docker images查看本地所有镜像

容器

查看容器

  • 使用docker ps -a查看所有的容器
  • 使用docker start [NAME]启动已经停止的容器

停止容器

  • docker stop <ID>
  • 重启使用restart

进入容器

  • 当容器在后台执行的时候，我们可以通过docker attatch <ID>进入后台执行的容器
  • exec当进入容器后输入exit后会退出容器，但是并不会停止容器

删除容器

  • docker rm -f <ID>

查看容器日志

  • docker logs -f <ID>查看容器内部的标准输出
